Overview

Relationship Between the Two Services

A web server delivers the website itself, while a file server provides the external files that the website uses and displays.

1

Web Server

The web server sends the HTML, CSS, and JavaScript to the browser so the website can appear on screen.

2

File Server

The file server stores assets such as images, text files, PDFs, and other downloadable content.

3

Public File URLs

The website uses public URLs from the file hosting provider to fetch and display those hosted resources.

4

Combined Output

The result is one complete webpage where the page and the files come from two different hosting services.
