# Deployment-ready website: Web server + File server relationship

## Recommended setup
- Web hosting: GitHub Pages
- File hosting: Dropbox or Cloudinary

## Files included
- index.html
- style.css
- script.js
- sample-file.txt

## What to edit before deployment
1. In `index.html`, replace the section placeholder with your actual section.
2. Upload an image and one file to Dropbox or Cloudinary.
3. Copy the public URLs.
4. In `script.js`, replace:
   - `hostedImageUrl`
   - `hostedFileUrl`
5. Deploy all website files to GitHub Pages.

## Notes
- The site UI already includes the student's name and a section placeholder.
- Once deployed on GitHub Pages, the site stays online as long as the repository remains published.
